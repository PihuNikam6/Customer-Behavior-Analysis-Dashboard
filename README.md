# Customer-Behavior-Analysis-Dashboard
This project focuses on analyzing retail sales data using Microsoft Excel to understand:

Revenue performance

Customer purchasing behavior

Category-wise sales contribution

Discount patterns

The final output is an interactive Excel dashboard with slicers, charts, and KPIs to support business decision-making.

# Objectives

Analyze customer purchase patterns and total spending

Compare category-wise performance (Beauty, Clothing, Electronics, Home Decor, Kitchen)

Track discount usage and impact

Create an interactive, visually appealing Excel dashboard

# Tools & Excel Features Used

Pivot Tables

Pivot Charts (Bar, Line, Pie)

Slicers & Filters

Conditional Formatting

Data Cleaning (removing duplicates, formatting, normalization)

Excel Formulas (SUMIF, COUNTIF, etc.)
